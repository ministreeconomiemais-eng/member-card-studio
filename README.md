# Member Card Studio

Application web de generation de cartes PVC pour plusieurs organisations.

## Deploiement Vercel

- Framework preset: `Other`
- Build command: `npm run build`
- Output directory: `.`

## Lancement local

```bash
npm start
```

Puis ouvrir:

```text
http://127.0.0.1:4173/
```

## Comptes de demo

- Super admin: PIN `0000`
- Les organisations ont leurs propres PIN dans l'application.
